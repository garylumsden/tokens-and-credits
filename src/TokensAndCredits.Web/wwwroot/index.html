How tokenisation works

A language model never reads letters or words directly. Your text is first chopped into tokens — small chunks (whole words, word-pieces, single characters, even emoji fragments). This split is done locally by a fixed dictionary, so it is deterministic, not AI: the same text always tokenises the same way.
